Visual Studio Code 扩展收集 ​

美化 ​

  1. Material Icon Theme - 美化文件图标。
  2. Github Theme - Github 风格主题。
  3. ReUI - React 风格主题。

Markdown ​

  1. Markdown all in one - Markdown 所需的一切（键盘快捷键、目录、自动预览等）。
  2. markdownlint - MD 语法检查提示。
  3. PlantUML - 增加 PlantUML 支持。
  4. Markdown Preview Mermaid Support - 增加 Mermaid 和流程图支持。
  5. Pangu-Markdown - 盘古之白 for Markdown，优化中文排版。
  6. Markdown Preview Github Styling - 将自带的 MD 渲染预览样式更改为 Github 的渲染样式。
  7. Markdown Preview Enhanced - 增加全新的渲染预览视图，图片助手。

日志 ​

  1. VSCode Journal - 快速创建日志（日记）。
  2. vscode-journal-view - 配合上面的快速创建日志插件使用，为侧边栏增加日志视图，方便浏览。

画图 ​

  1. Draw.io Integration - Draw.io 离线版，使用 *.drawio.svg 或 *.drawio.png 后缀名创建文件，即可进行编辑／使用（无需导出）。
  2. Excalidraw - Excalidraw 离线版，使用 *.excalidraw.svg 后缀名创建文件，即可进行编辑／使用（无需导出）。

Web 开发 ​

  1. Auto Rename Tag - 帮你自动重命名成对的 HTML/XML 标记，提高效率。

其他 ​

  1. 中文标点符号转换 - 将用户输入的中文标点符号自动更改为英文标点符号。
  2. Dictionary Completion - 英文单词补全。
  3. Word Count CJK - 中文字数统计。
  4. Office Viewer - 支持预览常见的 office 文件格式。
  5. Local History - 每次修改文件时，都会在本地保留包含旧内容的历史记录。
  6. Google Translate - 在编辑器中提供 Google 翻译，无需秘钥。

蛇精病 ​

  1. 坤坤鼓励师 - 在 VS Code 中连续写代码一小时（时间可配置），会有蔡徐坤专属篮球舞提醒你该休息啦！
